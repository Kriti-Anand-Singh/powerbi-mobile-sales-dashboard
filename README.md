# 📊 Mobile Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an **interactive Mobile Sales Analysis Dashboard** created using **Microsoft Power BI**. The dashboard helps analyze sales performance, customer behavior, payment methods, and brand performance across different cities and time periods.

The objective of this project is to transform raw sales data into meaningful insights through interactive visualizations and business intelligence techniques.

---

## 📈 Key Dashboard Metrics

* **Total Sales:** 769M
* **Total Quantity Sold:** 19K
* **Total Transactions:** 4K
* **Average Sales Value:** 40K

---

## 📊 Dashboard Features

* City-wise sales distribution using **Map Visualization**
* **Monthly sales trend** analysis
* **Customer ratings** analysis
* **Payment method distribution** (UPI, Debit Card, Cash, Credit Card)
* **Brand-wise sales performance**
* **Day-wise sales trend analysis**

---

## 🛠 Tools & Technologies Used

* **Microsoft Power BI**
* **Power Query** (Data Cleaning & Transformation)
* **Data Modeling**
* **DAX (Data Analysis Expressions)**
* **Data Visualization**

---

## 💡 Key Insights

* Sales vary across different cities and time periods.
* **Digital payment methods** like UPI and cards dominate most transactions.
* Some mobile brands consistently perform better in both sales and quantity.
* Monthly trends help identify peak sales periods.
